## Setup

<Info>
  You will need:

  - A [published bot](/learn/get-started/quick-start)
  - A [Hunter account](https://hunter.io/users/sign_up)
</Info>

<Steps>
  <Step title="Install the integration in Botpress">
    First, install the integration to your bot:

    1. In Botpress Studio, select **Explore Hub** in the upper-right corner.
    2. Search for the **Hunter** integration, then select **Install Integration**.

    In the **Configuration** menu, you should see one field: **API key**. Leave it empty for nowâ€”you'll come back to it after getting your Hunter API key.
  </Step>
  <Step title="Get your Hunter API key">
    Next, get your Canny API key:

    1. [Log in to your Hunter dashboard](https://hunter.io/users/sign_in).
    2. From the left sidebar, go to **API**.
    4. Copy your API key.
  </Step>
  <Step title="Configure the integration in Botpress">
    Now, you can finish setting up the integration in Botpress:

    1. In the integration's **API Key** field, paste your Hunter API key.
    2. Select **Save Configuration**.
  </Step>
</Steps>

<Check>
  Your Hunter integration is ready. You can use the integration's [Cards](#cards) to interact with your Hunter leads.
</Check>
