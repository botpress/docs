title: Update variables outside Studio
---

You can update your bot's [variables](/learn/reference/variables/overview) from outside Botpress Studio using the [Runtime API](/api-reference/runtime-api/getting-started).

<Info>
  You will need:

  - A [published bot](/learn/get-started/quick-start)
  - At least one declared [variable](/learn/reference/variables/overview)
  - [Authentication](/api-reference/authentication) for the Runtime API
  - Familiarity with JavaScript
</Info>

## Step 3: Update the variable using the API

Now, you can query the Runtime API's [`callAction`](/api-reference/runtime-api/openapi/callAction) endpoint to update the variable. You can make an HTTP request or use the [official Botpress client](/api-reference/runtime-api/getting-started#using-the-official-typescript-client):

<Tabs>
  <Tab title="Using an HTTP request">

    <CodeGroup>

    ```shell cURL
    curl --request POST \
      --url https://api.botpress.cloud/v1/chat/actions \
      --header 'Authorization: Bearer <your-token>' \
      --header 'Content-Type: application/json' \
      --header 'x-bot-id: <api-key>' \
      --data '{
      "input": {
        "newValue": "This is the new value of the variable"
      }
    }'
    ```

    ```python Python
    import requests

    url = "https://api.botpress.cloud/v1/chat/actions"

    payload = { "input": { "newValue": "This is the new value of the variable" } }
    headers = {
        "Authorization": "Bearer <your-token>",
        "x-bot-id": "<api-key>",
        "Content-Type": "application/json"
    }

    response = requests.post(url, json=payload, headers=headers)

    print(response.json())
    ```

    ```js JavaScript
    const url = 'https://api.botpress.cloud/v1/chat/actions';
    const options = {
      method: 'POST',
      headers: {
        Authorization: 'Bearer <your-token>',
        'x-bot-id': '<api-key>',
        'Content-Type': 'application/json'
      },
      body: '{"input":{"newValue":"This is the new value of the variable"}}'
    };

    try {
      const response = await fetch(url, options);
      const data = await response.json();
      console.log(data);
    } catch (error) {
      console.error(error);
    }
    ```
