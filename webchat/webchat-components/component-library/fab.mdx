The `Fab` (Floating Action Button) component is typically used to toggle the visibility of the Webchat UI in your application.

<Img src="../../assets/webchat-components_component-library_fab.png" />

### Usage

```tsx
import { Fab } from '@botpress/webchat'

function App() {
  const [isWebchatOpen, setIsWebchatOpen] = useState(false)

  return <Fab onClick={() => setIsWebchatOpen(!isWebchatOpen)} />
}
```

### Props

<ResponseField name="onClick" type="MouseEventHandler<HTMLButtonElement>">
  A callback function triggered when the Fab is clicked. Commonly used to toggle the Webchat visibility.
</ResponseField>
